# Pedestrian Detection using HOG in Python

## Overview
A complete pedestrian detection system using Histogram of Oriented Gradients (HOG) feature extraction and Support Vector Machine (SVM) classifier.

## Technologies Used
- Python 3.11
- OpenCV
- scikit-learn
- scikit-image
- Matplotlib
- Jupyter Notebook

## Methodology
1. Data Collection
2. Image Preprocessing (resize to 64×128, grayscale)
3. HOG Feature Extraction
4. SVM Model Training
5. Hyperparameter Tuning (GridSearchCV)
6. Evaluation (Precision, Recall, F1-Score)
7. Sliding Window Detection

## Results
- Test Accuracy: 100%
- Precision: 1.000
- Recall: 1.000
- F1-Score: 1.000
